# calculator
Personal project with The Odin Project

This calculator handles:
	- operations with two operands:
		- addition
		- multiplication
		- subtraction
		- division
	- operations with one operand:
		- percentage
		- square
		- square root
		- factorial
		- modulo
		
Available buttons:
	- 10 digits (0 to 9)
	- decimal point (.)
	
	- 4 basic operators (+ - × ÷)
	
	- percentage (%)
	- square (x²)
	- square root (√)
	- factorial (!)
	- modulo (MOD)
	
	- equals (=)
	
	- All Clear (AC)
	- Clear (C)

		
The calculator starts with an empty screen
